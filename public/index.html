TheRaffler

TheRaffler helps Meetup organizers to raffle off their schwag to event attendees.

Organizers input the Meetup event, and TheRaffler retrieves the RSVP list and display the attendees in a random order.

TheRaffler is

  • a Rails app
  • completely Open Source (MIT License)
  • repo'ed on Github - https://github.com/boscomonkey/theraffler/
  • tracked on Pivotal Tracker - https://www.pivotaltracker.com/projects/737639
  • deployed on Heroku - http://theraffler.herokuapp.com/
